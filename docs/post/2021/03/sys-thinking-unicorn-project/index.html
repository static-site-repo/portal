從系統思考角度讀《獨角獸專案》

William Yeh
2021-03-28
約 1348 字 - 預計閱讀 3 分鐘
course management

活動名稱：從系統思考角度讀《獨角獸專案》

時間：2021-11-24 (三) / 13:30–15:00 (因疫情影響延期舉辦)

會議：DevOpsDays Taipei 2021 的 603 會議室

課程簡介

《獨角獸專案》是本寫實刻劃軟體研發者日常甘苦的商業小說，但內行看門道，這本奇書，並不止於小說，而是真正可以照表操課的，以敏捷的 IT 能力驅動並翻轉整間公司的競爭力。

我在本書推薦序也曾如此建議：『請以「系統與組織的五大理念」觀點細讀，從頭分析故事的人事時地物及因果關係。經此一番系統思考，比起許多愛談高大上理念的敏捷、DevOps、數位轉型書籍，你將會有更深刻具體的領悟。』

問題是：如何系統思考？如何照表操課？

這場 90 分鐘的迷你工作坊，將以「系統思考」角度，帶你分析書中倡議的「五大理念」，你將會得到有助於形塑適合 IT 滋長企業文化的施力線索。

目標聽眾

已有《獨角獸專案》一書，想從中擷取可行洞見的人。

程度分級 ★☆☆ (通識)：介紹新知給尚未具備或僅具備少許該領域知識的人，也包含非技術議題。

聽眾收穫

認識高效能研發團隊的核心特質，以及促成這些特質的關鍵施力點。

注意事項

學員需知

  • 上課前，請先完成「課前作業」（後詳）。

  • 上課時，請攜帶「課前作業」及《獨角獸專案》一書。建議攜帶繁體中文版，以備隨時照頁數檢索。

課前作業

根據學員不同的狀況，我將課前作業分成兩部分：

  • 必備作業：所有人都務必在課前確實完成。
  • 選配作業：行有餘力，請在課前盡量做。做得越多，課堂上就越輕鬆。

必備作業

請依照「速讀」及「精讀」兩階段依序進行：

➊ 速讀階段：速讀完《獨角獸專案》至少一遍，先享受高潮起伏的劇情，並熟悉人物與關係。

➋ 精讀階段：挑出書中所有與「系統與組織的五大理念」有關的段落，越多越好。

選配作業

若行有餘力，請針對前面挑出的「五大理念」段落，分析其中的因果關係。
