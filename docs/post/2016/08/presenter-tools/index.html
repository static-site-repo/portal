簡報者的小工具

2016-08-22
tech
約 747 字 預計閱讀 2 分鐘 | 阅读

自從去年一月看了 ihower 的〈講個秘訣：技術簡報的好用工具〉之後，我才想到，身為 Mac 用戶，我好像太疏於好好整頓自己的電腦，讓我在簡報時更順暢。於是，經過一番嘗試，我也建構出自己的工具組合。

後來，在演講或講課時，總會遇到有人問道：「william，你是用怎麼弄出 xxx 效果的？」每次簡報，屢試不爽；乾脆自己整理一篇文章好了。

註：此文只列出我最常用的工具，更完整的，請見 ihower 的〈講個秘訣：技術簡報的好用工具〉。

布幕

Desktop Curtain，像舞台的暗紅色簾幕一樣，把雜七雜八的背景遮住，留下乾淨的背景給前景主角發揮。簡報時，我都會用這軟體把不相干的電腦桌面圖示遮起來，讓投影出來的畫面更乾淨，也有助於錄出更具專業感的影片。

實例：〈第二式：Life cycle - 開機、登入登出、關機 (18:20)〉這段影片，就用了這款 Desktop Curtain 軟體，讓 live demo 畫面更乾淨。

聚光燈

Mouseposé，像舞台的聚光燈一樣，把聚光燈打在想凸顯的地方，不重要的地方則弄暗。這軟體也能夠將你按的按鍵放大顯示出來，讓大家方便照著輸入；不過我比較少用到這一招。

實例：〈從限制理論看 DevOps〉演講第 47 分鐘開始，就用了這款 Mouseposé 軟體，讓大家容易聚焦在當下的重點區域。

放大鏡

Zoom It，其實是我在 Windows 時代就很愛用的軟體。像放大鏡一樣，局部放大重點區域。

實例：呃，暫時還沒有錄影。因為這一招，我比較常用在時間較長的課堂上，不常用在時間緊湊的演講。改天如果我會直接在演講會場，赤裸裸秀出 GitHub 或 Sublime Text 的程式片段，或許就會順便錄下來做記錄吧。

倒數計時器

Alinof Timer，在進行某些限時活動，或中場休息時，有這個東西，很好用。

實例：〈有了 Agile，為什麼還要有 DevOps？〉演講第 28 分鐘開始，就用了這款 Alinof Timer，作為倒數計時之用。



以上僅列出四個我在簡報時最常用的工具，給大家參考。
