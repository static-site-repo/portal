求職，別忘了突出你的亮點

2020-01-08

在網路上看到 Joe 喟然嘆曰：「今天重要功課：仔細看過明天 20 位履歷課同學的履歷以及繳交的功課。說起來這堂課好像是我們最耗能的一堂課，完全是做功德的一堂課。」

我，身為履歷表價值鏈另一端的面試官，也很能瞭解這番滋味。1

近年來，履歷教戰守則廣傳，有專書，甚至還有專門課程。理論上，「不懂得 包裝美化履歷 正確呈現履歷」的低級錯誤應該會越來越少見——其實不然。驚訝的是，即使是獵頭轉來的履歷也常無法倖免，真不知該怎麼說了。

遇到比較多的反例，是沒有亮點。

年資，不必然是亮點

先破除一個迷思：年資，不必然是忠誠，不必然是專注，不必然是匠人精神。一言以蔽之，不必然是亮點。

猶記 2016 年擔任軟體工程師面試官時，看著其中某一份履歷，我寫下如此感觸：

收到一份履歷，他在某接案公司待了 N 年。

很可惜。在我眼中，這只值兩年呀。

對接案公司無不敬之意。但是，字裡行間，實在看不出反駁我對他「只有兩年的工作經驗，只不過重複了 N 年」認知錯誤的理由。

其實，多年以來，每一段面試高峰期都會出現同樣的觀察與感觸——看不出求職者的亮點，尤其是與職務相關的亮點，遑論職涯層次的亮點。

書面的履歷表也就罷了。更麻煩的是，怕誤判，約來面試，試著口頭引導一下，都不見得引導得出來。

面試官當下的任務，是替用人單位選才，而不是當應徵者的職涯諮詢者。

請自己先正確表述你的亮點，而不是徒留年資流水帳。

成長性

不同企業、不同職種，側重的亮點或許不盡相同。

在我所處的軟體研發領域，「成長性」是非常重要的。除非某職缺有立即上戰場的急迫性，否則，我會側重在求職者的成長性，包括硬技能成長性及軟技能成長性；有時候，成長性甚至會重於即戰力。

什麼樣的成長性才稱得上是亮點？

且讓我套用裘凱宇老師在【過好人生學】課程所講的「站位／靶位」概念來分析：

  • 固定站位：在單一領域，發展到學徒、創造、達人的哪一個階段？
  • 固定靶位：在單一領域，持續創造了什麼成果？
  • 固定站位，不固定靶位：多元發展
  • 不固定站位，固定靶位：跨界創意
  • 不固定站位，不固定靶位：多元發展 ＋ 跨界創意

站位／靶位的固定與否，是個人的主觀選擇（或被動的被選擇）；不過，不管是屬於哪一種排列組合，請試著客觀回顧履歷一遍，能否據以勾勒出某些清晰的移動路徑？

成長性，從這路徑就看得出來了。

你，是一直在打怪升級成長 2，是在死守四行倉庫，還是在宇宙漂流？

敏銳於遊戲規則

履歷，畢竟是對既有產品（「個人」）既成事實的推銷；說句老實話，不免有為時已晚的風險。更可靠的，應該是在走到這田地之前，就下意識讓自己在站位／靶位兩方面都逐漸成長。

成長，不是臨時抱佛腳一蹴可幾，更不是刻舟求劍緣木求魚。

成長，需要敏銳於職場的遊戲規則。

其一：

其實，每一個位子都有可能發光發熱，重點仍在於是否有自覺進行「換位思考」及「系統思考」。

有自覺，你的一年工作經驗，就有別人 N 年的價值；反之，你的 N 年工作經驗，就只有一年的價值。

   — Quote: 2016-08-20 Tweet

其二：

不是在大公司就一定學得深，不是在新創就一定學得廣。

重點仍在於自己。在於「自己有沒有系統思考及換位思考的自覺及行動」。

在於專業態度。

「專業技能」固然重要，但是，能夠讓自己從流水帳的制式履歷表欄位中脫穎而出的，終究是「專業態度」。很準。只要針對每一份履歷表，個別客製化追問一兩次，幾乎就能分辨出哪些人有下意識追求「專業態度」的成長。

— Quote: 〈專業態度的養成〉

其三：

許多事情，跟你身在大公司還是小公司是低度相關的。心智框架，是要靠自己主動觀察、取經、思考的。

   — Quote: 2016-08-15 Tweet

成長，是下意識的選擇，是用持續的意志去磨銳遊戲規則的基本功：換位思考、系統思考、專業態度、心智框架。

求人找亮點？

大環境長期低薪，促使許多人尋求更多元的選項。就連「副業」這個老詞彙，居然也孳生「斜槓」這個聽起來更新潮、也更容易炒作的詞彙。職涯、生涯的重定位，一時之間蔚為風潮。

是真需求也好，是炒作起來的需求也好，有需求，就有供給。於是乎，坊間有了《做自己的生命設計師》這樣的書，有了所謂【設計你的人生】工作坊，甚至像 S 姐在〈默默的，你認知的這些工作都已轉型〉所說，開始有了 “career coach” 這樣的新興職業。3

打著 “career coach” 名號的，在這一兩年，真的如雨後春筍（尤其是在 FB & LinkedIn）。

多試試，無妨；但，請慎選建議。畢竟，自稱 “coach” 者，並不總是 coach。

正派的推薦

其實我真的很鼓勵求職者，在丟出履歷、赴約面試之前，先讀讀一些正派的相關書籍，或尋求正派的諮詢者。畢竟，不是每位面試官都想在緊湊的當下順便兼任求職諮詢。

面試官很累的，別奢想他們都是卞和。請自己打磨出你的亮點。

四處征戰的應徵者也是很累的，我知道。所以，請不要奢想你是當下就能七步成詩感動天地的曹植，請事先就打磨出你的亮點。

 

之所以會強調「正派」二字，是因為⋯⋯在某些書上看到一些耍小聰明的招數。說實在的，那些小聰明，請不要秀出來～～

書籍

推薦這幾本正派的書。不管是身為面試者還是被面試者，我都從中獲得許多啟發：

  • 《但願你因工作而閃亮》

  • 《2030 轉職地圖》

  • 《三年後，你的工作還在嗎？》

  • 《工作愈換愈好，得有這些本事》

  • 《轉職必勝班》

對外商或國際化環境感興趣的，除了《2030 轉職地圖》之外，可再參考以下這本：

  • 《外商、大企業求職秘笈》
